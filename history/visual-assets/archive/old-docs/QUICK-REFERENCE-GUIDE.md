## Maintenance Quick Reference

### To update a visual:

1. Regenerate PNG with same filename
2. Replace file in `book-source/static/img/part-N/chapter-NN/`
3. Verify markdown reference still accurate
4. Check alt text still describes content
5. Test rendering in Docusaurus

1. Generate PNG using Visual Skills Framework v5.1.0
2. Save to appropriate chapter directory
3. Create embedding entry in placement map

6. Update visual asset inventory
