## Summary

Redesign the RoboLearn landing page to establish a distinctive "Industrial-Kinetic Futurism" visual identity that embodies Physical AI & Robotics education. Transform from generic template aesthetics to a cohesive design language featuring:

- Dark industrial foundation (#141418 background)
- Electric cyan primary (#00d4ff) with amber/orange accents (#ff9500)
- JetBrains Mono + Syne typography
- Orchestrated page load animations with scroll-triggered reveals
- Blueprint aesthetics with animated grid backgrounds
- Custom geometric components replacing stock Docusaurus elements

**Technical Approach**: Component-based redesign using React 19 + TypeScript 5.6 + CSS Modules within Docusaurus 3.9.2 framework. CSS-only animations (no Framer Motion dependency). Performance-optimized with Lighthouse targets (>= 90 performance, >= 95 accessibility).

**Intersection Observer Hook**:

**File**: `src/hooks/useScrollReveal.ts`
```typescript
import { useEffect, useRef } from 'react';

export const useScrollReveal = (options = {}) => {
  const ref = useRef<HTMLElement>(null);

  useEffect(() => {
    const element = ref.current;
    if (!element) return;

    const observer = new IntersectionObserver(
      ([entry]) => {
        if (entry.isIntersecting) {
          element.classList.add('visible');
          observer.disconnect(); // Performance optimization
        }
      },
      { threshold: 0.1, ...options }
    );

    observer.observe(element);
    return () => observer.disconnect();
  }, [options]);

  return ref;
};
```
